# 📡 NetAssist AI
NetAssist AI — Intelligent Telecom Support Powered by RAG. 📡🤖
**NetAssist AI** is an AI-powered telecom support assistant that helps users solve network and service-related issues. 🤖

It uses a **RAG (Retrieval-Augmented Generation)** system with **ChromaDB** to search information from **FAQs, PDF documents, and CSV files** and provide accurate, context-aware answers. 📚🔍

The project is designed to make customer support faster and more efficient by understanding natural language queries and retrieving the most relevant information from its knowledge base. ⚡💬

It is lightweight, scalable, and can be easily extended with new data sources. 🚀

---

## ✨ Features

- 🤖 AI-powered telecom support assistant
- 📄 Supports FAQs, PDF documents, and CSV files
- 🔍 Semantic search using ChromaDB
- 💬 Natural language question answering
- ⚡ Fast and lightweight
- 🔄 Easy to update with new knowledge sources
- 🚀 Built with a modern RAG architecture

---

## 🛠️ Tech Stack

- 🐍 Python
- 🎨 Streamlit
- 🗄️ ChromaDB
- 🧠 Sentence Transformers (Embeddings)
- 🤖 Ollama / Local LLM
- 🔗 LangChain *(if used)*

---

## 🚀 How It Works

1. 📥 Load FAQs, PDFs, and CSV files.
2. 🧠 Generate embeddings for the documents.
3. 🗄️ Store embeddings in ChromaDB.
4. 💬 User asks a question.
5. 🔍 The system retrieves the most relevant information.
6. 🤖 The AI generates an accurate answer based on the retrieved context.
